# unmarshal-bounty

## Telegram Bot

### Build and Run

```
# install dependencies
pip install -r requirements.txt

# set env
export TOKEN={BOT-TOKEN}
export MODE={MODE:dev or prod}
export UNMARSHAL_API_KEY={API-KEY}

# run
python3 main.py
```
